** Разложение sin(x) в ряд Маклорена

*** Используемые формулы

Необходимо найти n-нный член ряда.

Вычислим производную n-ого порядка:

\[(\sin{x})' = \cos{x}\]

\[(\sin{x})'' = -\sin{x}\]

\[(\sin{x})''' = -\cos{x}\]

\[(\sin{x})'''' = \sin{x}\]

\[(\sin{x})^{(n)} = \sin{(x + \frac{\pi n}{2})}\]

Найдём n-нный член разложения:

\[q_n = \frac{(-1)^n x^{2n + 1}}{(2n + 1)!}\]

Выведем c:

\[c = \frac{q_n}{q_{n - 1}} = \frac{-x^2}{2n(2n + 1)}\]

И выделим начальный член q_0:

\[q_0 = \frac{(-1)^0 x^{0 + 1}}{(0 + 1)!} = x\]

** Разложение cos(x) в ряд Маклорена

*** Используемые формулы

Вычислим производную n-ого порядка:

\[(\cos{x})' = -\sin{x}\]

\[(\cos{x})'' = -\cos{x}\]

\[(\cos{x})''' = \sin{x}\]

\[(\cos{x})'''' = \cos{x}\]

\[(\cos{x})^{(n)} = \cos{(x + \frac{\pi n}{2})}\]

Найдём n-нный член разложения:

\[q_n = \frac{(-1)^n x^{2n}}{(2n)!}\]

Выведем c:

\[c = \frac{q_n}{q_{n - 1}} = \frac{-x^2}{2n(2n - 1)}\]

И выделим начальный член q_0:

\[q_0 = \frac{(-1)^0 x^0}{0!} = 1\]

** Разложение ln(x) в ряд Маклорена

*** Используемые формулы

Применим следующее разложение[fn:1: Источник: http://www.math.com/tables/expansion/log.htm]:

\[\ln{x} = 2\sum_{n = 0}^{\infty}{\frac{1}{2n + 1}\left(\frac{x - 1}{x + 1}\right)^{2n + 1}}\]

Найдём c:

\[c = \frac{q_n}{q_{n - 1}} = \frac{(2n - 1) \left(\frac{x - 1}{x + 1}\right)^2}{2n + 1}\]

И выделим начальный член q_0:

\[q_0 = \frac{x + 1}{x - 1}\]

Функция натурального логарифма существует для значений \(x > 0\).
